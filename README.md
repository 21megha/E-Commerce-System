# E-Commerce-System
E-commerce System with Product hierarchy (Electronics, Clothing, Books) using inheritance and polymorphism, ShoppingCart functionality, and order management.

Setup Instructions

Follow these steps to set up and run the project locally:

1️⃣ Prerequisites

Java Development Kit (JDK 8 or above)

Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or Command Line

Git (optional, for cloning the repository)

2️⃣ Clone the Repository (Optional)
git clone https://github.com/21megha/sql-datawarehouse-project


(Replace with your actual e-commerce project repository link if different)

3️⃣ Project Structure

Ensure the folder structure is organised as follows:

ecommerce/
├── products/
├── cart/
├── orders/
├── customer/
└── ECommerceSystem.java


Each package contains related classes to maintain modularity and readability.

4️⃣ Compile the Project

From the project root directory:

javac ecommerce/**/*.java

5️⃣ Run the Application
java ECommerceSystem
